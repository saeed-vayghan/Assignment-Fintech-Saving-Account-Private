# Alborz Bank: Savings Account Architecture Explorer

A high-fidelity, interactive architectural presentation system designed to document and explore the technical specifications of a modern fintech savings platform.

## 🚀 Project Overview
This project serves as a comprehensive technical submission for the Alborz Bank architectural assignment. It organizes **57+ architectural artifacts** into a structured, 6-phase journey, covering everything from initial business requirements to complex distributed systems design and cloud infrastructure.

### Key Features
- **Hierarchical Navigation**: 6 distinct phases of architectural discovery.
- **Interactive Diagrams**: Real-time rendering of Mermaid.js diagrams with high-fidelity **Zoom & Pan** (via `svg-pan-zoom`).
- **Fullscreen Deep-Dives**: Edge-to-edge rendering for complex architectural landscapes.
- **Midnight Obsidian Theme**: A premium, high-contrast dark mode designed for professional presentations.
- **Micro-Artifacts**: Deep linkage to detailed markdown specifications for every service and entity.

## 🛠️ How to Run
Due to browser security policies regarding local file fetching (CORS), this project **must be served via a web server**.

### 1. Requirements
- **Python 3.x** (installed by default on most modern systems).
- A modern web browser (Chrome, Firefox, Safari).

### 2. Execution
Open your terminal in the project root and run:
```bash
git clone https://github.com/saeed-vayghan/Assignment-Fintech-Saving-Account-Private.git

cd Assignment-Fintech-Saving-Account-Private

python3 -m http.server 2026
```

### 3. Access
Navigate to the following URL in your browser:
[http://localhost:2026/presentation.html](http://localhost:2026/presentation.html)
